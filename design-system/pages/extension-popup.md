# Extension Popup Override

## Purpose

Popup 是 quick pulse launcher，不是完整工作台，也不是设置页。

## Override Rules

1. 第一块必须是 `Pulse summary`
   - `Open assignments`
   - `Due soon`
   - `Unseen updates`
   - `Blocked sites`
2. 第二块是 `Fast actions`
   - `Open assistant`
   - `Quick export`
   - `Settings/Auth`
3. `Quick export` 作为第三块，不再独占页面
4. 小尺寸下只允许单列

## Copy Rules

- 用 `Quick pulse`
- 强调这是体温计和发射台，不是完整操作台
