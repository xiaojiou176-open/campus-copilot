# Code of Conduct

## Our Commitment

We want Campus Copilot to be a repository where contributors can discuss ideas, report issues, and review changes without being attacked, mocked, or pushed out of the conversation.

## Expected Behavior

- Be respectful in technical disagreement.
- Critique code and decisions, not people.
- Be honest about uncertainty and limits.
- Prefer evidence over posturing.
- Keep sensitive security details out of public threads.

## Unacceptable Behavior

- Harassment, abuse, or personal attacks
- Deliberate intimidation or exclusion
- Publishing private or sensitive information without consent
- Repeatedly derailing collaboration with bad-faith behavior

## Reporting

For conduct concerns, contact the repository maintainer through the repository owner profile:

- <https://github.com/xiaojiou176-open>

For security-sensitive concerns, use the process in [SECURITY.md](SECURITY.md) instead of a public discussion thread.
